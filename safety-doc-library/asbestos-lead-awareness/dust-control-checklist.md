## Checklist
Verify dust controls for dust-generating work (silica, lead, wood, general).

| # | Item | Yes | No | N/A | Notes |
|---|---|:--:|:--:|:--:|---|
| 1 | Source controls selected (water/wet methods or tool-mounted HEPA vacuum) |  |  |  |  |
| 2 | Containment/enclosure where needed to limit spread |  |  |  |  |
| 3 | Negative pressure/HEPA filtration if required by the task |  |  |  |  |
| 4 | Bystanders and occupants kept clear of the dust zone |  |  |  |  |
