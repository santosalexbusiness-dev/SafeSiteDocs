## Checklist
Verify before using a mobile (rolling) scaffold.

| # | Item | Pass | Fail | N/A | Notes |
|---|---|:--:|:--:|:--:|---|
| 1 | Casters lock and are in good condition; locked during use |  |  |  |  |
| 2 | Height-to-base ratio within safe limits (outriggers if required) |  |  |  |  |
| 3 | Fully planked with guardrails |  |  |  |  |

| 5 | Floor is level, firm, and free of holes/obstructions |  |  |  |  |
