## Checklist
Inspect an extension ladder before use.

| # | Item | Pass | Fail | N/A | Notes |
|---|---|:--:|:--:|:--:|---|
| 1 | Rails free of cracks, bends, or splits |  |  |  |  |
| 2 | Rungs tight, not bent or slippery |  |  |  |  |
| 3 | Rung locks/dogs engage properly |  |  |  |  |
| 4 | Rope and pulley intact and functional |  |  |  |  |
| 5 | Feet/safety shoes present and not worn |  |  |  |  |
