## Checklist
Verify hearing protection in noisy areas.

| # | Item | Pass | Fail | N/A | Notes |
|---|---|:--:|:--:|:--:|---|
| 1 | Noise level warrants protection (raise voice at arm's length = protect) |  |  |  |  |
| 2 | Protection rated adequately for the noise |  |  |  |  |
| 3 | Earplugs inserted correctly / muffs seal fully |  |  |  |  |
