## How to Use This Template
Complete a JHA **before** a task, ideally with the crew that will do the work. Break the job into steps, identify the hazards in each step, then list the controls that eliminate or reduce each hazard. Review it on site, adjust if conditions change, and have the crew sign on. Use the hierarchy of controls — try to **eliminate** or **substitute** the hazard first, then use **engineering controls**, then **administrative controls / safe work practices**, and use **PPE** as the last layer.
